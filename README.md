# 🏺 Zohar - Oráculo Visual Inmersivo

**Zohar** no es solo una aplicación web; es una experiencia de introspección digital diseñada para transformar la curiosidad en iluminación. A través de una arquitectura minimalista y un diseño de alto impacto visual, Zohar entrega mensajes de sabiduría mística envueltos en una atmósfera de "Lujo Digital".

---

## ✨ Características Principales
* **Arquitectura Monolítica:** Desarrollado íntegramente en HTML5, CSS3 y JavaScript puro. Sin dependencias externas, optimizado para carga instantánea.
* **Refinamiento Visual Dinámico:** La interfaz reacciona al mensaje entregado, cambiando la atmósfera de la app mediante transiciones de luz y color (Efecto "Vaso de Oro").
* **Curación de Contenido:** Mensajes basados en principios de sabiduría antigua, adaptados para el usuario moderno.
* **Diseño Mobile-First:** Optimizado para ser consultado desde cualquier dispositivo mediante escaneo de códigos QR.

## 🛠️ Tecnologías
* **HTML5:** Estructura semántica.
* **CSS3 (Custom Properties & Animations):** Motor de renderizado visual y efectos de "polvo de estrellas".
* **JavaScript (Async/Fetch):** Lógica desacoplada mediante archivos JSON para facilitar la escalabilidad del contenido.

## 🚀 Propuesta de Valor (Business Logic)
Este proyecto está diseñado bajo el modelo de **Micro-SaaS**. Es ideal para:
1.  **Hostelería:** Entretenimiento premium para comensales en espera.
2.  **Wellness B2B:** Integración en plataformas de meditación o tiendas esotéricas.
3.  **Marca Blanca:** Capacidad de personalización visual (estilos oro, profundo, místico) para diferentes nichos.
